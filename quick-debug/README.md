## Setup (Run This First!)
```bash
cd /mnt/c/Users/jdabl/SyllabAI/quick-debug

```

## Run the Test
```bash
npm test
```

## What It Does
1. Opens your site in a browser (you can watch!)
2. Checks what components are loaded
3. Monitors API calls
4. Handles Google login (you do it manually)
5. Tests file upload
6. Takes screenshots
7. Shows you everything that happened
